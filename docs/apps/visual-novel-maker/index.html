メモ:

  • ストア版(非Steam版)ではインストールできるのは1台(RPGツクールMVのみ2台)らしい(オフィシャルFAQ)。だとすると、複数の環境で開発する場合にはSteam版の方が良いのかもしれない。

    • …と思いきや、 ストア版はSteamキーが付属しているのでストアで買えばSteam版として使用できるらしい。

  • Steam版には Live2D が付属していないエディションがある。

  • ラノゲツクールとラノゲツクールMVは別物。
