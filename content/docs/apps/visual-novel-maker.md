# リンク

## 基本情報

- [オフィシャルサイト](https://tkool.jp/lngmv/)

  - [オフィシャルのDL版](https://tkool.jp/lngmv/download/index.html)

- [Steam](http://store.steampowered.com/app/495480/Visual_Novel_Maker/)

メモ:

- ストア版(非Steam版)ではインストールできるのは1台(RPGツクールMVのみ2台)らしい([オフィシャルFAQ](https://tkool.jp/support/faq/activation))。だとすると、複数の環境で開発する場合にはSteam版の方が良いのかもしれない。

  - …と思いきや、 [ストア版はSteamキーが付属している](https://forum.tkool.jp/index.php?threads/%E3%83%A9%E3%83%8E%E3%82%B2%E3%83%84%E3%82%AF%E3%83%BC%E3%83%ABmv%E3%81%AE%E3%82%B2%E3%83%BC%E3%83%A0%E3%83%91%E3%83%83%E3%82%B1%E3%83%BC%E3%82%B8%E3%81%AE%E4%BD%9C%E6%88%90%E3%81%8C%E3%81%A7%E3%81%8D%E3%81%AA%E3%81%84%E7%97%87%E7%8A%B6.578/)のでストアで買えばSteam版として使用できるらしい。

- Steam版には Live2D が付属していないエディションがある。

- [ラノゲツクール](https://tkool.jp/lng/)とラノゲツクールMVは別物。

# 初期設定/序盤知識

## 日本語

Tools \> Preferences \> Application \> General の Language

設定が終わったら再起動。 なお、https://tkool.jp/support/faq/lngmv/lngmv_faq\[オフィシャルのFAQ\]によると、すぐに再起動するのでなく、終了後10秒ほど待ってから起動すると良いらしい。

## マニュアル

ヘルプメニューから開けるが、一瞬どう見ればいいのかわからない。

左端のナビゲーションの中の Table of Contents をクリックしてメニューを出した後、そのメニューの中の▲をクリックすれば全体の目次が表示される。
